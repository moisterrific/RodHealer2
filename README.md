# RodHealer2
Heals players when using the Rod of Discord 

Originally made by [AncientGods](https://github.com/ancientgods/RodHealer), updated by me.

**TShock version:** 4.4.0 pre-11

**API version:** 2.1

## How to Install
1. Put the .dll into the `\ServerPlugins\` folder.
2. Grant your desired group(s) the `rodhealer.ignore.damage` permission.
3. Enjoy spamming RoD without consequence!

### Permission
`rodhealer.ignore.damage`

## How to Use
Just use Rod lol you won't lose HP anymore.
